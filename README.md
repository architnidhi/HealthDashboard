# HealthDashboard

An Android application that replicates a rehabilitation/therapy dashboard with adaptive layouts for both smartphones and 10.4-inch tablets.

## Features
- 📊 **Adaptive UI** - Automatic layout switching between phone and tablet (10.4-inch)
- 🔥 **Firebase Integration** - Real-time data synchronization with dummy data
- 📈 **Interactive Charts** - Movement data and history graphs using MPAndroidChart
- 📄 **PDF Generation** - Export dashboard data as comprehensive PDF reports
- 📱 **Material Design** - Clean, professional interface following Material Design guidelines

## Tech Stack
- Kotlin
- Firebase Realtime Database
- MPAndroidChart
- iText PDF Library
- Material Design Components

## Setup Instructions
1. Clone this repository
2. Open in Android Studio (Flamingo+ recommended)
3. Add your `google-services.json` from Firebase console
4. Build and run

## Firebase Setup
- Create project in [Firebase Console](https://console.firebase.google.com/)
- Register app with package name: `com.example.healthdashboard`
- Enable Realtime Database
- Dummy data auto-generates on first launch
